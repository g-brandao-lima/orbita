Replace the stub in app/services/signal_service.py with the full implementation. Follow the RESEARCH.md code examples closely. Key implementation details:

1. **detect_signals(db, snapshot)** orchestrator function:
   - Get previous snapshot via _get_previous_snapshot(db, snapshot)
   - Call all 4 detector functions
   - Filter duplicates via _is_duplicate(db, signal)
   - db.add() and db.commit() only new signals
   - Return list of new signals

2. **_get_previous_snapshot(db, current)**: Query FlightSnapshot filtered by route_group_id, origin, destination, departure_date, return_date, id < current.id, order by collected_at DESC, limit 1.

3. **_booking_classes_to_dict(booking_classes)**: Convert list of BookingClassSnapshot to dict {class_code: min_seats} using MIN across OUTBOUND/INBOUND for same class (the bottleneck approach per RESEARCH.md).

4. **_check_balde_fechando(current, previous)**: If previous is None, return None. Build dicts for both. Check classes K and Q. If prev_seats >= 3 and curr_seats <= 1, return DetectedSignal with signal_type="BALDE_FECHANDO", urgency="ALTA", details=f"Classe {class_code}: {prev_seats} -> {curr_seats} assentos".

5. **_check_balde_reaberto(current, previous)**: If previous is None, return None. Check all classes. If prev_seats == 0 and curr_seats > 0, collect reopened classes. If any, return DetectedSignal with signal_type="BALDE_REABERTO", urgency="MAXIMA".

6. **_check_preco_abaixo_historico(db, snapshot)**: If price_classification != "LOW", return None. Use subquery to get avg price of last 14 snapshots of same route. If count < 3, return None. If snapshot.price < avg_price, return DetectedSignal with signal_type="PRECO_ABAIXO_HISTORICO", urgency="MEDIA".

7. **_check_janela_otima(snapshot)**: Calculate days_until = (departure_date - date.today()).days. If <= 0, return None. Determine domestic via BRAZILIAN_AIRPORTS set (hardcoded ~25 airports: GRU, CGH, VCP, GIG, SDU, BSB, CNF, PLU, SSA, REC, FOR, POA, CWB, FLN, BEL, MAO, NAT, MCZ, VIX, CGB, GYN, SLZ, THE, AJU, JPA, PMW, IGU). Window: domestic (21, 90), international (30, 120). Return DetectedSignal with signal_type="JANELA_OTIMA", urgency="MEDIA".

8. **_is_duplicate(db, signal)**: Query DetectedSignal with matching route_group_id, origin, destination, departure_date, return_date, signal_type where detected_at >= (datetime.now(timezone.utc) - timedelta(hours=12)). Return True if exists.

9. **_get_avg_price_last_n(db, snapshot, n=14)**: Use subquery pattern from RESEARCH.md. Return (avg_price, count).

IMPORTANT per RESEARCH.md Anti-Patterns:
- Use datetime.now(timezone.utc) consistently (NOT datetime.utcnow() which is deprecated)
- Use db.query() style (legacy) to match existing codebase, NOT select() style
- Detector functions are pure (no db.add inside them), only detect_signals orchestrator does db.add/commit
- Handle previous=None gracefully in balde detectors
